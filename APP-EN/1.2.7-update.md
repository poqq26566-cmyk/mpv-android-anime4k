## Changelog (v1.2.6 → v1.2.7)

### 🆕 New Features

1. **Tree View Folder Browser**
   - Added tree view mode for folder browsing

2. **WebDAV Auto-load Subtitles**
   - Subtitles with matching filenames are now automatically loaded when playing WebDAV files

### 🔧 Improvements & Optimizations

- **.nomedia Rule Optimization** — Improved algorithm, refined warning text, added warning dialog when enabling reverse scanning options
- **Upscale Configuration Optimization** — Rolled back old upscale algorithms and files, added official recommended presets, introduced quality tiers (Smooth / Balanced / High Quality) for different device performance levels
- **Gesture Control Refinements** — Added 8% dead zone on the bottom, top, and right edges of the screen to prevent accidental touches
- **WebDAV File Icon Display** — Improved file icons in WebDAV browser

### 🧹 Code Cleanup

- Removed unused resource files and code
