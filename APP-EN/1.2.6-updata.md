## Changelog (v1.2.5 → v1.2.6)

### 🆕 New Features

1. **Background Playback (Listen Only)**
   - Added "Listen Only" option in the "More" menu
   - Video continues playing in the background, audio only
   - Uses a foreground Service to keep the process alive
   - Service automatically stops when returning to the foreground

2. **Decoder Presets**
   - Added MPV decoder preset switching in Playback Settings
   - Presets: Fast / Default / High Quality / GPU HQ / Low Latency / SW Fast
   - Each preset corresponds to different performance/quality levels

3. **Device Info Page**
   - New device information feature
   - Displays HDR support, 10-bit encoding, Dolby Vision Profiles
   - Lists all hardware-supported video/audio encoders

4. **Auto Orientation Detection**
   - Portrait-shot videos automatically switch to portrait mode when playing

5. **Real-time Download Speed Display**
   - Shows real-time download speed on the right side of the screen during online playback

6. **Chapter Display on Progress Bar**
   - Added toggle in settings to enable/disable chapter display on the progress bar
   - Shows chapter markers on the progress bar when enabled
   - Displays the current chapter name above the progress bar
   - Chapter menu refactored into a right-side drawer

7. **Remaining Time Display**
   - Click the time text on the right side of the progress bar to toggle between elapsed and remaining time

8. **Folder Scanning Rules**
   - Added `.nomedia` rules toggle
   - Added scan hidden folders toggle
   - Reorganized into a new settings section

9. **Alternative Download Button**
   - Added "Alternative Download" button in the update dialog
   - Primary link points to Bilibili article, alternative to Tencent Docs

10. **Volume Control Optimization**
    - Added "Control System Volume" toggle in settings
    - Option to keep volume changes after exit or restore previous volume

11. **WebDAV Account Editing**
    - Added edit button for saved WebDAV accounts

### 🔧 Bug Fixes

- Fixed issue where the forward button was not working
- Fixed WebDAV links requiring authentication but failing to play
- Fixed WebDAV crashing on some devices
- Fixed DanDanPlay service failure
- Fixed online subtitle search failure
- Fixed Release build failing to parse Bilibili series
- Fixed control component frame drops when upscaling mode is enabled

### 📦 Other Changes

- Optimized video file scanning performance
- Improved WebDAV account management UI
- Updated documentation links
