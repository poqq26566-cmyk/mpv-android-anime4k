# Update Notes

## Version Notice

Although the app version remains 1.2.3, this build includes significant updates. The version number was kept at 1.2.3 (instead of incrementing to 1.2.4) to maintain version synchronization with the Chinese version, making it easier to manage unified updates in the future.

## What's New

### New Features
- **Network URL Playback**: Added support for playing videos from network URLs
- **Screen Orientation Toggle**: Added the ability to switch between portrait and landscape modes

### Improvements
- **WebDAV Account Interface**: Optimized the WebDAV account management interface
- **Drawer Navigation**: Improved the right-side drawer navigation experience

### Bug Fixes
- **English Localization**: Fixed remaining Chinese text that was not properly translated to English
- **Default Language**: The app now defaults to English interface regardless of system language settings
- **Playback History**: Fixed a bug in the playback history feature

---

**Note**: This version focuses on providing a complete English user experience and enhanced playback capabilities.
