# Version 1.2.4 Update Notes

## New Features

1. **Custom Speed List**: Added customizable playback speed list covering MPV player's supported speed range with 0.25x step increments

2. **Playback Speed Memory**: The app now remembers your last used playback speed

3. **Direct Video View Option**: Added option in settings to directly display video view, bypassing folder list

4. **MediaInfo Integration**: Integrated MediaInfo library - now you can retrieve detailed video information

5. **File Operations**: Added in-app file/folder management capabilities (rename/delete/copy)

## UI Optimizations

6. **Settings Organization**: Consolidated Bilibili video download, danmaku download, and subtitle download into a unified category in settings

7. **Bilibili Login Page**: Updated back icon, implemented immersive status bar at top, optimized QR code prompt text with left alignment

8. **Bilibili Anime Playback Page**: Updated back icon, replaced gradient background with solid pink background

9. **Homepage Enhancements**: Implemented immersive status bar at top, bottom plus menu now auto-avoids navigation bar, video list background changed to pure white

10. **Video List Page**: Duration and file size labels now aligned to thumbnail bottom, video titles aligned to top with reduced font size, maximum 3 lines of title displayed

## Bug Fixes & Optimizations

11. **Volume Bar Fix**: Removed percentage symbol from playback interface to resolve volume bar glitch on certain devices

12. **English Version Improvements**: Completed translation for previously untranslated Chinese text throughout the app
